Typical Usage
^^^^^^^^^^^^^

Step 0: Load python version 2.7.13.

.. code:: bash

    $ module load python/2.7.13

Step 1: Plot your figure.

.. code:: bash

    $ HemTools [subcmd] -f input_data

You can always see all available sub-commands by:

.. code:: bash

    $ HemTools -h

Report bug
^^^^^^^^^^   

Most visualizations are done interactively. Error message will be displayed directly on your screen. Please copy those error msgs and email me, Yichao.Li@stjude.org.

.. note:: `HemTools report_bug` will not work in most data visualization subcmds.
